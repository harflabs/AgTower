## [1.0.0] - 2026-05-29

Initial public release.

- Attention-triage queue across multiple repositories and agent sessions.
- Zero-config Claude Code and Codex session discovery with live terminals
  (xterm.js + WebGL).
- Kanban dashboard, split-pane monitoring, and keyboard-first navigation.
- SQLite-backed session persistence with restart recovery.
- macOS native chrome (invisible toolbar, vibrancy), system tray, and a
  signed auto-updater.
